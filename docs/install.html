download

Pre-built binaries are attached to each GitHub release. Pick your platform:

platformbinary
macOS (Apple Silicon) tccp-macos-arm64
Linux (x64) tccp-linux-x64
Windows (x64) tccp-windows-x64.exe

install (macOS / Linux)

# Download the binary for your platform
curl -Lo tccp https://github.com/sethlupo/tccp/releases/latest/download/tccp-macos-arm64
chmod +x tccp
sudo mv tccp /usr/local/bin/

# Verify
tccp --version

install (Windows)

# Download tccp-windows-x64.exe from the releases page
# Move it somewhere on your PATH, or run it directly
tccp.exe --version

first-time setup

# Store your Tufts cluster credentials
tccp setup

This prompts for your Tufts username, password, and email. Credentials are stored in the system keychain (macOS) or ~/.tccp/credentials (Linux/Windows).

build from source

git clone https://github.com/sethlupo/tccp.git
cd tccp
cmake -B build -S . \
  -DCMAKE_TOOLCHAIN_FILE=vcpkg/scripts/buildsystems/vcpkg.cmake \
  -DVCPKG_TARGET_TRIPLET=arm64-osx \
  -DCMAKE_BUILD_TYPE=Release
cmake --build build
./build/tccp --version

Dependencies (managed by vcpkg): libssh2, yaml-cpp, fmt, CLI11, replxx.
